# Weather API Fetcher

## Description

A Python application that fetches real-time weather data from a public API and displays it in a user-friendly format.

## Features

- Fetches weather data from API
- Accepts city input from user
- Displays temperature and wind speed
- Handles API errors

## Technologies Used

- Python
- Requests Library
- Open-Meteo API


## How to Run

Install dependency:

pip install requests


Run:

python weather_api.py
